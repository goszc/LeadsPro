## 🚀 Como Baixar e Instalar

O LeadsPro foi compilado como um único arquivo executável seguro para Windows (Setup).

1. **Faça o Download:** Acesse a página de [Releases](../../releases/latest) no canto direito e baixe o arquivo `LeadsPro Setup 1.0.0.exe`.
2. **Instale:** Duplo clique no executável. Ele instalará automaticamente e criará um atalho na sua Área de Trabalho.
3. **Crie sua Licença:** Ao abrir o aplicativo, insira sua Chave de Acesso para ativar a sua máquina e começar a usar.

## ⚡ Funcionalidades Principais

- **Extrator Turbo do Maps:** Raspe centenas de empresas em segundos buscando por palavra-chave ou região. Dados extraídos incluem Telefone, Nome, Categoria, Latitude/Longitude e Website.
- **Automação Headless de WhatsApp:** Motor embutido invisível. Leia o QR Code uma vez e dispare campanhas em massa diretamente pelo App para os Leads capturados sem precisar abrir o navegador.
- **Multithreading Nativo:** Arquitetura inteligente de Paralelismo. O sistema cria "trabalhadores" (Workers) simultâneos no seu computador para varrer o mapa 10x mais rápido.
- **Filtros e Exportação:** Explore os leads capturados na tabela interativa e exporte os dados consolidados para CSV, JSON ou XLSX (Excel).
- **Sem Risco de Bloqueio:** Rotaciona perfis silenciosamente evitando alertas rígidos de Scraper do Google Maps.
